
### 第一节：介绍 Milvus  

#### 1. Milvus 的发音  
Milvus 的发音是 **/ˈmil vəs/**，中文可读作“密尔沃斯”。  

#### 2. 应用场景  
- **RAG（检索增强生成）**  
- **对话机器人**  
- **门头招牌图片识别**  
- **声纹识别**  

#### 3. 向量数据库与向量计算库对比  
- **Faiss**：高性能计算引擎，适用于纯向量计算加速，但需搭配存储系统。  
- **Elasticsearch (ES)**：性能强大，功能复杂。  
- **Milvus**：全功能向量数据库，适用于大规模生产环境，支持 GPU 加速。  
- **ChromaDB**：开发友好，适合轻量级应用和快速迭代。  
- **Pinecone**：闭源。  

#### 4. Milvus 版本分类  
- **Lite**：精简版（开发版）。  
- **Standalone**：单机版。  
- **Distributed**：分布式版本。  

#### 5. 核心功能  
- **建表（Collections）**  
- **插入数据**  
- **检索数据**  
- **删除数据**
